About Us!

We're successful developers and software engineers from Lebanon who invented a bunch of success repositories in a place filled with obstacles and economic crises!

Lebanon Stars is a simple project created by Ali Alachkar and with the help of contributors using Angular 17, Tailwind, and DaisyUI. The main purpose of the project was to add all the Lebanese repos to be viewed by other developers, university students, or even recruiters!
